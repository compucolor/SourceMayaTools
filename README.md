# SourceMayaTools
A plugin for Maya 2025 and newer for exporting models and animations to Source.
Based on [CoDMayaTools](https://github.com/TheEnbyWitch/CoDMayaTools)

# Features
 - Exporting models and animations to *.SMD files
 - Supports subtracting animation data
 
# Requirements
 - [Autodesk Maya 2025 or newer](http://autodesk.com/maya)
